# 📝 Task Manager Web App

## 📌 Objective
A full-stack Task Management System that allows users to manage tasks, track progress, and view analytics.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based)
- 📝 Task Management (Create, Read, Update, Delete)
- ✅ Mark tasks as completed
- 🔍 Filtering (Status & Priority)
- 🔎 Search by Title
- 📊 Analytics Dashboard (Total, Completed, Pending)
- 🎨 Clean UI with Light & Dark theme sections

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
